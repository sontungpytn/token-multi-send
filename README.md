# token-multi-send smart contract

A boilerplate hardhat project with some defaults.

- [solhint][1]
- [eslint][6]
- [prettier][2]
- [typescript][4]
- [typechain][3]
- [dotenv][7]

## Prerequisities

- [Node.js][5]

## Install

    yarn

## Tasks

Build the project

    yarn run build

Run tests

    yarn test

Lint solidity and typescript code

    yarn run lint

Check file formatting

    yarn run lint

Generate code coverage

    yarn run coverage
